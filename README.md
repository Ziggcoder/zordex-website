# ZordexAI Website

Futuristic React + Vite landing page for ZordexAI, an AI-powered PCB review and electronics services brand.

## Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Structure

```text
src/
  assets/
  components/
  pages/
  sections/
  styles/
```
